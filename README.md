# Web Note Pad

  PHP demo web site to store notes in SQLite.

  Used for testing and demonstrations.


## Requirements

   * apache2; tested with 2.4.25-3+deb9u7
   * php7.0; tested with 7.0.33-0+deb9u3
   * sqlite3; tested with 3.16.2-5+deb9u1
   * php7.0-sqlite3; tested with 7.0.33-0+deb9u3
